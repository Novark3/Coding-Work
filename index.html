Coding Basics

HTML

HTML is a very important coding device that helps design wepages and amounts of text. HTML was developed by CERN and it allowed for easy webpage designing and editing. HTML allows people like me or you who might not have much experience with coding, to jump into it and make very detailed and nicely arranged websites. HTML allows us to insert pictures, organize ideas, and present things in a visually pleasing way. With the correct code, ideas become clear and information becomes easier to understand.

CSS

CSS is another type of coding that allows users to make large-scale attribute changes with only a few lines of code. This saves a developer lots of time and energy. Not having to go through adding hundreds of attributes, but instead having them on a single line, helps coders tremendously. CSS also allows for further in-depth changes such as borders, buttons, and text alignment without a scramble of messy code.

Tags

Tags are HTML's way of determining how to interpret a message. The tag tells the software whether it is a paragraph or an image. Tags help the software interpret the listed information correctly and efficiently. They categorize information into ver seperate and distinct bins so that the program does not waste time sorting through all of the different pieces of text and color individually. This way, the entire process off coding and rendering is sped up tremendously. Some very important tags are the:

Click For More CSS
  • Paragraph tag
  • Heading tag
  • Doctype tag
  • Image tag

The paragraph tag looks like < p > . This tag allows for paragraphs to be sectioned off and organized in a very clear way. Paragraphs allow for the separation of ideas and information.

The heading tag looks like < h1 > and can have a number anywhere from 1-6. These numbers give priorities if there are many different titles in close proximity to one another. This prevents glitches and messy graphical overlays.

The doctype tag tells the software what kind of document the listed document is. In this paper, the doctype is html. The tag is listed in the code as < !DOCTYPE html >. This allows for proper reading and displaying of the document. Any document that is written in HTML has this tag at the very beginning of the coded section.

The last tag that I will be talking about is the image tag. This tag allows the user to place images on the page for the reader to see. The tag is listed as < img > in the code. The way that someone would use this tag would be by writing out the tag, and then adding the src attribute to it. After adding the url to the quoted section, the image is displayed for all to see.

Attributes

Attributes provide the software a further level of interpretation and give the coder another level of personalization and customization, like a modifier. Attributes allow the coder to change the font a paragraph is shown as, the color of the text, it allows for indents, and you can also change the size of the font and much more. All of these different tools allow for amazing websites with stunning visuals. Each tag has its own set of attributes, some more than otehrs, there are no limits of how many attributes are allowed per tag. This allows for hundreds, maybe even thousands, of different possibilities for one paragraph. Some possible attributes are:

  • src
  • href
  • style
  • font-family
  • text-indent
  • font-size
